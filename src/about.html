What's New!

Jan ##, 2022: Version 2.0.1:

Simplified navigation by removing Main page and having user go directly to active Assessment page after splash screen.

Dec 25, 2021, Version 2.1

Implemented tabbed help information for About, About RSRAFVP, About Security, About Privacy pages.  Home button introduced.  What's New modal created to provide better notification of changes.  Navigation bar and menu simplified to a single object for consistent behavior and functionality across all pages.

Dec 16, 2021, Version 2.0

Implemented basic authentication functionality (login) for Firebase and Google providers.  Implemented initial saving and recalling an assessment.

Nov 16, 2021, Version 1.0.1

Fixed date print on PDF.

Oct 31, 2021, Version 1.0

All basic pages with navigation implemented.  Guest functionality for creating, resetting, printing, and exporting an assessment.
